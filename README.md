# HoodBook Mini (THDA)

Booking web app for The Hood Dance Academy.  
Includes WhatsApp checkout, QR check-in, and an Admin scanner. PWA-ready.

## Run locally
```bash
npm install
npm run dev
```

## Deploy (Vercel)
- Import this repo in Vercel
- Framework: **Vite + React**
- Deploy → open URL → Add to Home Screen (iPhone)
